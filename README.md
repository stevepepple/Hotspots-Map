React + Redux example of map.

### Run the client
The react code for the web and progressive mobile app is in client/src. It's a project 'create-react-app' boilerplate. 

    cd client
    npm install
    npm run start

    open http://localhost:3000/events

### Run the server

    npm install
    npm run start
